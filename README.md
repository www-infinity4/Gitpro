# Gitpro — GP Suite 🛸

**Gitpro** is the Quantum Mechanic AI in the **GP Suite** — a family of four intelligent GitHub companions. Users pull in their repos and work on them with Gitpro's quantum mechanic suggestions: top-level page content edits, styling, design ideas, and assimilations.

## GP Suite AI Family

| Agent | Symbol | Nickname | Role |
|-------|--------|----------|------|
| **Gitpal** | 🔮 | Ace | Palace AI — floating orb companion, suggests pinned content & professional moves |
| **Gitpub** | 📡 | Pubby | Publisher AI — checks repos are live, emits file signals, builds with tools |
| **Gitpro** | 🛸 | Pro | **Quantum Mechanic** — content edits, styling, design ideas, assimilations *(built here)* |
| **Gitpin** | ∆ | Pin | Vector Triangulator — pins and maps exact content coordinates across repos |

## Gitpro Features

- **Quantum Scan** — select any connected repo and Gitpro runs a multi-stage analysis across content, styling, design, performance and assimilation dimensions
- **Actionable Suggestions** — each suggestion includes priority (CRITICAL → LOW), the target file, a copyable code snippet, and one-click Apply
- **Type Filters** — filter suggestions by ✏️ Content, 🎨 Styling, 🖼️ Design, 🔗 Assimilation, ⚡ Performance
- **Ask Pro Chat** — conversational interface backed by Gitpro's quantum mechanic persona; quick-reply prompts to get started fast
- **Re-scan** — re-run the quantum scan at any time to refresh suggestions
- **Agent Switcher** — sidebar to navigate between all four GP Suite agents

## Tech Stack

- [Next.js 16](https://nextjs.org/) (App Router, TypeScript)
- [Tailwind CSS v4](https://tailwindcss.com/)
- React 19

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
